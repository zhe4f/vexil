# Vexil

> 多连接并发 · 断点续传 · 流式校验 · TLS 加密 · 跨平台

Vexil 是一个高性能的局域网文件传输工具。通过多连接并发和滑动窗口等优化技术，榨干你的网络带宽。

---

## 为什么选 Vexil

- **快** — 4-16 条 TCP 连接并发传输，千兆网跑满 110+ MB/s
- **稳** — 自动保存传输进度，SHA-256 流式校验拒绝传完等待
- **安全** — 内置 TLS 加密，保护数据传输
- **简** — 单文件可执行程序，下载即用，无需安装
- **美** — 暗色/亮色双主题，设计简约无广告

## 快速开始

双击打开，点「发送」或「接收」。

首次启动会显示使用指南：

- 📤 **发送** — 选择文件 → 选择设备 → 开始发送
- 📥 **接收** — 设置端口和目录 → 等待连接
- 🔄 支持断点续传，关闭后自动恢复
- 🔒 内置 TLS 加密，加密传输内容
- 🌓 右上角切换亮色/暗色主题
- ⚙ 右上角齿轮进入设置
