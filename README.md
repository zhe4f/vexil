# Vexil

> 多连接并发 · 断点续传 · TLS 加密 · 跨平台

Vexil 是一个高性能的局域网文件传输工具。通过多连接并发和滑动窗口算法，榨干你的网络带宽。

---

## 为什么选 Vexil

- **快** — 4-16 条 TCP 连接并发传输，千兆网跑满 110+ MB/s
- **稳** — 断点续传 + SHA-256 校验，断电也不丢进度
- **安全** — 内置 TLS 加密，保护数据传输
- **简** — 单文件可执行程序，下载即用，无需安装
- **美** — 暗色/亮色双主题，设计简约无广告
